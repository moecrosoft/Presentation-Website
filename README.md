# HDB Resale Prices Analysis & Prediction

A Streamlit web app that presents the analysis and prediction of HDB resale prices in Singapore.  
It showcases **data cleaning, EDA, feature engineering, and machine learning models** with interactive visualizations such as heatmaps, histograms, and bar charts.  
Key insights include the impact of **floor area, lease attributes, accessibility, and year of sale** on resale prices.
